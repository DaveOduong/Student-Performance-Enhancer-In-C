This program is an automated grade-curving tool that dynamically recalculates student scores using Z-Score Standardization—the standard method used in professional grading.

Instead of adding a flat, arbitrary bonus, it evaluates how far each student scored above or below the actual class average, then remaps the entire class onto a fresh target average and spread that you define.

How It Works in 3 Quick Steps
1. Gathers Data: It lets you test using your preset scores (27, 35, 56, 44) or type in an entirely new batch of custom student grades.

2. Analyzes the Class: It calculates the current class performance by finding the exact mathematical mean (average) and the standard deviation (how spread out the scores are).

3. Applies the Curve: It mathematically adjusts every student's score to fit your desired new average. It also contains built-in safety rules to prevent runtime crashes (like division by zero if everyone got the same grade) and automatically caps final scores cleanly between 0% and 100%.
